Form Controls

Radio buttons

Radio buttons allow users to see all available choices at once and select exactly one option.

<div class="form-check form-check-inline">
	<input type="radio" id="inlineRadioOptions" name="inlineRadio1" class="form-check-input" value="option1">
	<label class="form-check-label" for="inlineRadio1">1</label>
</div>
When to use

When users need to select only one option out of a set of mutually exclusive choices.

When to use something else
  • Use checkboxes if there are multiple possible selections to be made.
  • Use a dropdown if there are too many options to list and insufficient space.

Search fields

Search capabilities will be available in Dockets and Parties to reduce user scrolling time.

<input class="form-control" type="text" aria-label="Filter">
When to use

Use search forms when there is an abundance of content, such as docket entries or many parties on a case.
