Basic Grid Structure

Grid systems play an important role in web design by ensuring a cohesive, structural layout. Grids allow developers to arrange content in a meaningful way while maintaining visual consistency.

Case view will be laid out using a 12 column grid.

The first, second, and third row will automatically stack on top of each other when the screen is less than 576px wide.
