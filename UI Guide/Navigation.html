Navigation

Navigation bars guide users beyond the default content, allows them to perform tasks, and orients them within the site.

Specifications

The top navigation bar should have a dark gray background (#444) for consistency and ease of recognition for help desk calls or other support inquiries.

The navigation icons will be right-aligned with USCourts and the name of the application being used to access the case summary left-aligned (in this case, Internal Case View).

Side navigation

The side navigation bar for Case View will reflect which version of Case View they are viewing. The purpose of this is to incorporate familiar elements from the current Case View application so they are able to recognize quickly that they may be in a view which the public does not have access.

Specifications

The side navigation bar will have the ability to expand or contract on hover. When contracted, only the icons for each section will be visible. When expanded, the icon with the section title will appear.

Side navigation will be fixed so it can be accessed at all times by the user.

When a cursor is on a section on the side navigation bar, the background should changed to light gray and text should change to black for visibility.

Guidance

  • Show users where they are within the menu options by highlighting the current section.
  • Present links in priority order.
  • Use descriptive, recognizable link labels - no jargon or unfamiliar terms.
