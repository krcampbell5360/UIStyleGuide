Headers

Court information header

Headings that include court contact and clerk information will be as follows:

  • All court information should be capitalized.
  • Include seal of the state of Arkansas left of the court name and address.
  • Clerk name should be right-aligned with all other content aligned to the left.

Case information header

Headings featuring general case information (case number, description, etc.) will be as follows:

  • The case description will be capitalized and in yellow/gold (#FCE205).

    Example:

    XXCR-XX-XXX - STATE V JOHN DOE

  • Title content will have the first letter of each word capitalized.
  • Content will be all capitalized.

Case information should use a dark grey background (#5B616B).

Position

The case information header should remain accessible when scrolling through the page.
