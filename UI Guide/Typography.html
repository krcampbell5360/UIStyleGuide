Typography

*Typography from Bootstrap

Typefaces

The default fonts from the Bootstrap library will be used for Case View. Bootstrap is a widely used component library which makes it familiar to end users and provides a clean, simple look and feel. Utilizing Bootstrap's default fonts also ensures efficiency in development.

Native font stack:

Default root font-size (16px) and line-height 1.5.

Default font-family is Helvetica Neue, Helvetica, Arial, sans-serif.

Specifications

Section titles, such as Parties, Dockets, etc., are coded using:

<span class="h3">...</span>

All field titles within each section, such as Case Status, Related Cases, etc., receive custom CSS:

.field-title {font-weight: 600;}

Typesetting

Line length

Maintaining a consistent line length increases the readability of content. The average line length is between 50 to 75 characters per line. The ideal length is 66 characters per line.

Alignment

To provide a natural starting point for the reader's eye, use typeset flush left unless specifically outlined elsewhere in the UI Style Guide.
