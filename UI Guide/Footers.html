Footers

Footers provide information useful to visitors that is not located anywhere else in navigation.

Accessibility

Code navigation so that pressing the tab key moves focus from link to link.

Typeface

Footers will have muted text.

This is muted footer text.
<div class="card text-center">
	<div class="card-footer text-muted">This is muted footer text.</div>
</div>

Usability

Guidance
  • Footer links should point to helpful content for users
  • Include branding information
  • Limit contact information to general email and/or phone numbers
  • Clickable links for phone numbers should be included for dialing from a mobile phone
