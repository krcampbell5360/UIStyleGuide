Modals

Modals are used sparingly. They are to be used to provide the user with more data/details without taking up too much space on the screen.

Modals are used in the following situations: to highlight history information and as a way to show Internal Court Connect users more details regarding a "MUST APPEAR" designation.

History modals

History information will be presented in a modal. Modal headers will be in adjective + noun format (i.e. "Case ID History").

Behavior

Modals will pop out when the history indicator is pressed.

Data layout

History data will be presented in the format indicated in the example below.

Example

		<div class="modal fade" id="caseStatusHistory" tabindex="-1" role="dialog" aria-labelledby="exampleModalLabel" aria-hidden="true">

      				<div class="modal-header">
