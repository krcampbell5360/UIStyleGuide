Icons

Icons help the user locate menu or navigation items quickly while saving space on a web page. Icons should be a clear representation of the content it replaces and should not clutter the page.

Icons for Case View are from the Material Icons library.

Accessibility

Icons should have a color contrast ratio of at lease 3:1. For specific information regarding icons, refer to each UI Component page.
