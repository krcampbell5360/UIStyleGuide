Overview

The UI Design Style Guide is a living document which will be amended over time. As design standards solidify, those decisions will be recorded here to maintain consistency across future projects.

The UI guide will provide component design standards and snippets of code for use by developers to facilitate efficiency.

Design

The UI design used to create the look and feel of the Case View for Modernization Project #1 comes from the Bootstrap library.

Accessibility

An important aspect of design is the consideration of all users. To that effect, all sites and pages created must adhere to the ADA accessibility requirements. Those guidelines can be found here.

Further accessibility guidelines can be found in its related component section.
